# Enterprise Employee Management System 🏢

Enterprise-grade Employee Management System built using .NET 8, React.js, SQL Server, Docker, and Microservices architecture.

## 🚀 Features
- JWT Authentication
- Role-Based Authorization
- Employee Management
- Department Management
- API Gateway
- Logging & Exception Handling
- SQL Optimization
- Docker Support

## 🏗 Architecture
- Clean Architecture
- CQRS Pattern
- Repository Pattern
- RESTful APIs
- Microservices Ready

## 🛠 Tech Stack

### Backend
- .NET 8
- ASP.NET Core Web API
- Entity Framework Core
- SQL Server

### Frontend
- React.js
- TypeScript
- Material UI

### DevOps
- Docker
- GitHub Actions

## ▶️ Setup Instructions

### Backend
```bash
cd backend
dotnet restore
dotnet run
```

### Frontend
```bash
cd frontend
npm install
npm start
```
